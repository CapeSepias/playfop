The Application

PlayFOP Labels is a sample Play Framework application for PlayFOP and Apache FOP. It demonstrates generating PDFs and images (PNGs) from Play Twirl templates.

PlayFOP Labels relies on:

  • Bootstrap, as provided by Adrian Hurtado's Play-Bootstrap
  • jQuery
  • Color emoji from Google's Noto font family

This is the Scala version (source code) of PlayFOP Labels. A Java version of PlayFOP Labels with equivalent functionality is also available.
